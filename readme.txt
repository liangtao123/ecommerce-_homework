index.html是首页 

有特惠价的出租
以及附近出租

在顶部导航栏的免费注册 以及 登陆 会分别跳转到login.html 以及 signup.html

点击搜索栏
显示出租的产品

任意点击一个产品 跳转到search.html 该页面显示了出租的产品以及相应的地点

任意选择一个产品 会跳转到item_detail.html 显示出租产品的押金、租金以及 交易地点
点击加入购物车 会跳转到car.html 
若此时点击 查看产品详情 会跳转回上一页item_detail.html
 若点击去购物车结算 则跳转至 pay.html 模拟扫码付款 点击支付成功 会跳转至success.html
则此时 交易完成。
